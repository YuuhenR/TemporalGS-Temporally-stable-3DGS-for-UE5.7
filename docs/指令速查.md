# TemporalGS 指令速查

> 所有 `r.TemporalGS.*` 和 `TemporalGS.*` 都在编辑器底部「输入控制台命令」框里敲,回车执行。

## 一、加载场景

```
TemporalGS.Load                         加载并自动缩放显示(默认 train_7000.ply)
TemporalGS.Load E:/路径/xxx.ply         加载指定 .ply
TemporalGS.Clear                        删除所有 splat actor
```
加载后:右上「大纲」点选 `GaussianSplatActor` → 鼠标移到视口 → 按 `F` 聚焦。

## 二、课题核心对比(ghosting 前后)——最重要

先确保 TSR 开着:
```
r.AntiAliasingMethod 4                  开启 TSR(4=TSR,2=TAA)
```
然后**一边连续转动/平移相机,一边切**:
```
r.TemporalGS.WriteVelocity 0            ← before:splat 运动时拖影/糊(问题)
r.TemporalGS.WriteVelocity 1            ← after :splat 运动时清晰(你的解)
```
> 这就是 D04 课题要证明的效果。若 `1` 反而更糊,说明 velocity 符号反了,告诉我。

## 三、视觉调节(运行时,免重编)

```
r.TemporalGS.Sigma 1.5                  splat 大小/锐利度(小=锐,大=糊),试 1.5~4
r.TemporalGS.OpacityScale 2             密度(填空隙),试 1~3
r.TemporalGS.Occlude 1                  场景深度遮挡(splat 被地形挡住),0=关
r.TemporalGS.Mode 1                     1=各向异性椭圆(真3DGS),0=圆盘后备
r.TemporalGS.Debug 1                    画成亮品红(排查可见性),0=真彩色
r.TemporalGS.Enable 1                   总开关
```
朝向不正:选中 Actor → 右侧 Details 的「旋转(Rotation)」直接转。

## 四、量化评测(可选,出数据)

沿同一相机轨迹分别录两段帧序列(`WriteVelocity 0` 和 `1`),然后:
```
python TemporalGS/Benchmark/metrics.py --baseline <0的目录> --fixed <1的目录>
```
输出 temporal flicker 降低百分比(越低越稳)。详见 `TemporalGS/Benchmark/README.md`。
